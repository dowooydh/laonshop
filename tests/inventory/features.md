# 주요 기능 목록

## 쇼핑 탐색

- 홈에서 남성/여성 상품 목록으로 진입
- 성별별 상품 목록 조회
- 상품 상세 조회
- 상품 이미지, 카테고리, 설명, 가격, 사이즈 표시

## 장바구니

- 상품 상세에서 사이즈와 수량 선택
- 장바구니 담기
- 바로 구매 시 장바구니 저장 후 `/cart` 이동
- 장바구니 수량 증감
- 장바구니 상품 삭제
- localStorage 기반 장바구니 유지
- 빈 장바구니 상태 표시

## 회원 인증

- 회원가입: 이름, 이메일, 비밀번호, 선택 전화번호
- 로그인: 이메일/비밀번호
- 로그아웃
- 로그인 필수 페이지 보호: checkout, mypage, order detail
- 삭제되거나 유효하지 않은 세션 사용자 처리

## 주문/결제

- 배송정보 입력
- 카드/간편결제 인증결제창 진입
- 계좌이체, 정기결제, 수기결제 비활성 표시
- 주문 생성 시 서버에서 상품 재조회와 금액 재계산
- 판매 종료 상품 포함 시 오류 처리
- KSPAY 결제창 폼 생성
- KSPAY callback 브릿지
- KSPAY result 서버승인
- 주문 상태: PENDING, PAID, FAILED, CANCELED
- 결제 완료 시 장바구니 비우기

## 마이페이지

- 사용자 정보 표시
- 최근 주문 50건 표시
- 주문 상태 배지
- 주문 상세 이동
- 주문 내역 빈 상태

## 카드사 심사 필수 요소

- 푸터 사업자정보
- 이용약관
- 개인정보처리방침
- 청약철회/교환/환불 안내
- 결제수단 경로 캡처 가능성
- KSPAY 결제창 진입 가능성

## 운영/보안 불변조건

- 금액은 정수 원 단위
- 결제 금액은 서버 계산값 기준
- 카드정보 비저장
- PG 응답 로그/증적에 민감정보 저장 금지
- 미확보 PG 스펙은 `NEEDS_PG_SPEC` 상태 유지
- 운영 상품/계정/설정 같은 마스터 데이터 삭제 금지
